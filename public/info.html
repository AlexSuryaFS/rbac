About This Application

A comprehensive Role-Based Access Control (RBAC) system built with modern web technologies

Demo Version: This is a static demo using localStorage for demonstration purposes.

Authentication

Secure user authentication with password hashing and session management

Role Management

Flexible role-based permissions with admin and user roles

Access Control

Protected routes and middleware-based authorization

Modern UI

Beautiful Bootstrap-based interface with responsive design

Technical Stack

Backend
  • Node.js & Express.js
  • Passport.js Authentication
  • Express Sessions
  • bcryptjs Password Hashing
Frontend
  • Bootstrap 5
  • Bootstrap Icons
  • Responsive Design
  • Modern CSS Features

Project Structure

Core Directories
  • /config - Database and configuration
  • /middleware - Authentication middleware
  • /routes - Express route handlers
  • /public - Static HTML files
Key Files
  • server.js - Main application entry
  • package.json - Dependencies and scripts
  • auth.js - Authentication routes
  • pages.js - Protected page routes
